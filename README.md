> 
> Тор, куб и икосфера с различными моделями освещения — от простого градиента до физически корректного ламбертова отражения с прожекторным конусом и затуханием по расстоянию.

---
О проекте

В учебных проектах часто используют встроенный OpenGL Lighting. Я поставила себе задачу **реализовать освещение полностью самостоятельно**, чтобы глубже понять:

- как работают матрицы трансформаций;
- как устроены модели освещения (Ламберт, затухание, прожектор);
- как выполняется сортировка по глубине без Z-буфера (painter's algorithm);
- как генерируются сложные поверхности (икосфера через subdivision).

**Весь код написан с нуля.** OpenGL используется только для вывода на экран — вся математика освещения и трансформаций реализована вручную.

Три проекта

1. Тор — градиентное освещение

**Задача:** Визуализировать тор с имитацией прожектора без использования OpenGL Lighting.

**Решение:**
- Параметрическая генерация поверхности (24×24 сегмента)
- Вращение с помощью клавиш WASD
- Освещение зависит от X-координаты центра треугольника на экране
- Цвет интерполируется между темно-красным и ярко-красным

**Ключевые параметры:**
- R1 = 0.8 (большой радиус тора)
- R2 = 0.25 (малый радиус трубки)
- stacks = 24, slices = 24

2. Куб — программный рендеринг

**Задача:** Реализовать полный софтверный 3D-пайплайн с правильной сортировкой по глубине.

**Решение:**
- 8 вершин, 6 граней
- Цепочка: масштабирование → вращение → перенос → проекция
- Painter's algorithm — сортировка граней по средней Z
- Каждая грань разбивается на 11 подтреугольников для плавного градиента
- Автовращение по всем осям

---

3. Икосфера — физически корректное освещение

**Задача:** Создать сферу с правильной геометрией и реалистичной моделью света.

**Решение:**

**Генерация геометрии:**
- Икосаэдр через золотое сечение
- Два уровня рекурсивного разбиения с проекцией на сферу
- Итог: 80 треугольников

**Модель освещения:**
1. **Ламбертово диффузное отражение** — яркость пропорциональна cos угла
2. **Прожекторный конус** — угол 40°
3. **Квадратичное затухание** — attenuation = 1 / (1 + 0.5 × dist²)

**Проекция:** Ортографическая
